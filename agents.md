# Контекст разработки AI-модуля анализа закупок

## Цель
Разработка модуля для подбора истории схожих закупок, формирования аналитической сводки и предсказания стратегий снижения цены.

## Требования к поиску схожих закупок
Поиск осуществляется по следующим признакам:
1. **Заказчик** - полное или частичное совпадение наименования организации
2. **Регион** - совпадение региона проведения закупки
3. **Вид работ** - совпадение по описанию объекта закупки
4. **Ключевые слова** - поиск по ключевым словам в описании
5. **Диапазон НМЦК** - начальная (максимальная) цена контракта в заданном диапазоне

## Аналитическая сводка
На основе найденных закупок формируется:
- количество найденных схожих закупок
- заказчик
- регион
- вид работ
- НМЦК (текущей закупки)
- средний процент снижения цены
- минимальное снижение
- максимальное снижение
- медианное снижение

## ML-модель предсказания снижения

### Признаки модели
1. **Текстовые признаки** (Bag of Words)
   - Токенизация описания закупки
   - Удаление стоп-слов
   - Топ-500 наиболее частых слов

2. **Категориальные признаки** (One-hot encoding)
   - Регион
   - Способ определения поставщика

3. **Числовые признаки**
   - Логарифм НМЦК (нормализованный)
   - OKPD2 код (первые 2 знака)

4. **Бинарные признаки**
   - Требуется обеспечение заявки
   - Ограничения для МСП

### Алгоритм
- Линейная регрессия с градиентным спуском
- L2-регуляризация
- Предсказание процента снижения от НМЦК

### Стратегии
- **МИНИМАЛЬНОЕ СНИЖЕНИЕ** (< 5%) - низкая конкуренция
- **УМЕРЕННОЕ СНИЖЕНИЕ** (5-15%) - средняя конкуренция
- **АКТИВНОЕ СНИЖЕНИЕ** (15-30%) - высокая конкуренция
- **АГРЕССИВНОЕ СНИЖЕНИЕ** (> 30%) - очень высокая конкуренция

## Структура данных входной записи
```json
{
    "reg_number": "номер закупки",
    "Организация, осуществляющая размещение": "заказчик",
    "Регион": "регион",
    "Наименование объекта закупки": "описание",
    "Начальная (максимальная) цена контракта": "НМЦК",
    "auction": [{"amount": "финальная цена"}],
    "reduction_percent": "процент снижения (для обучения)"
}
```

## API для использования

### Анализатор схожих закупок
```python
from procurement_ai import ProcurementAnalyzer

analyzer = ProcurementAnalyzer()
analyzer.add_records(historical_records)
similar = analyzer.find_similar(customer="...", region="...", work_type="...")
summary = analyzer.get_analytics_summary(similar)
```

### ML-предсказатель
```python
from procurement_ai import ReductionStrategyPredictor

predictor = ReductionStrategyPredictor()
predictor.train(training_data)  # данные с reduction_percent
result = predictor.predict(new_record)
print(result['predicted_reduction_percent'])
print(result['strategy'])
print(result['recommendations'])
```
